# 醒了吗

纯前端的极简饮酒安全兜底与清醒计时工具。

## 运行

直接用任意静态服务器打开当前目录即可，例如：

```bash
cd /Users/howell623/Projects/醒了吗?
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 当前实现范围

- 首次配置：性别、体重、个人酒量阈值、紧急联系人。
- 支持一键载入演示酒局，方便直接看交互和巡逻节奏。
- 酒局页：啤酒 / 红酒 / 白酒 / 洋酒四种卡片，支持 `+` / `-`。
- 首次记录某类酒时，可调整该酒类本次酒局的度数和单次容量。
- 公式：
  - 纯酒精 `= 容量(ml) × 度数(%) × 0.8`
  - 清醒倒计时 `= 纯酒精(g) ÷ 8g/h`
- 安全机制：
  - 达到阈值 50% 时，仅触发一次平安报备。
  - 超阈值且 30 分钟无操作，触发 60 秒预警拦截。
  - 超阈值且 60 分钟无操作，触发 60 秒紧急拦截。
  - 任意加减酒重置巡逻计时。
  - 点击“我已安全 / 结束酒局”后停止所有巡逻。
  - 跨天后自动清空当局，避免旧酒局残留。

## Web 版限制

- 浏览器不能真正“静默自动发短信”。当前实现会在触发时尝试唤起系统短信页并预填文案，同时复制短信内容。
- 页面被系统彻底关闭后，浏览器无法继续运行定时器。所以这是 Web MVP，不是原生后台守护版本。
- 所有数据仅保存在浏览器本地存储中，不接后端。
